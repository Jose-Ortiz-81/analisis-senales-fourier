# Simulación y Análisis de Señales con la Transformada de Fourier

## Descripción

Este proyecto fue desarrollado como parte de una actividad académica para analizar señales en el dominio del tiempo y de la frecuencia utilizando la Transformada de Fourier en Python.

Se implementaron tres señales básicas:

- Señal senoidal
- Pulso rectangular
- Función escalón

Posteriormente se aplicó la Transformada Rápida de Fourier (FFT) para analizar su comportamiento en el dominio de la frecuencia.

---

## Objetivos

- Generar señales básicas.
- Representarlas en el dominio del tiempo.
- Aplicar la Transformada de Fourier.
- Graficar la magnitud y la fase del espectro.
- Verificar las propiedades de la Transformada de Fourier.
- Comparar los resultados obtenidos.
